📋 1. Intake Summary — What I Heard
Team Context
New NAEF L&D team, currently being built. Support function serving the broader Amazon External Fulfillment network.
Primary Training Need
Leadership development training for Amazon leaders across the NAEF network. No existing structured program in place.
Build Constraint
Need to build quickly and at scale. Fast turnaround is a priority — team can't wait for a multi-month curriculum build.
Delivery Platform
Amazon Learn (internal LMS). Tracking completion and assessment scores is expected at a minimum.
❓ 2. Open Questions — Next Discovery Session
Question Why It Matters Priority
What specific leadership behaviors are breaking down on the floor right now? Defines the actual performance gap. Without this, we're building content, not solving problems. High
Who exactly is the learner? (New managers, experienced leaders, specific levels?) Audience determines scenario complexity, tone, and prior knowledge assumptions. High
What does "leadership training" mean to business stakeholders vs. the L&D team? Prevents misalignment between what gets built and what leadership expects. High
What's the target completion timeline for the first program? Determines whether we build a modular series, a single course, or a blended journey. Medium
Are there existing materials — job aids, SOPs, past trainings — we can build from? Reusing existing content significantly accelerates build time. Medium
Who will reinforce the learning on the job? Is manager support available? Behavior transfer requires on-the-job reinforcement. L3 measurement depends on this. Medium
🎯 3. Assumed Performance Gap
Working hypothesis (to be validated): Amazon leaders in the NAEF network are not consistently demonstrating key leadership behaviors — including giving direct feedback, managing team performance, and building trust — because they lack a structured, scalable development resource they can access on demand.
Current State
No structured leadership training program exists for the NAEF L&D team's audience. Development is ad hoc or reliant on informal manager guidance.
Desired State
Leaders can access bite-sized, behavior-focused training tied to Amazon's Leadership Principles, available on demand through Amazon Learn, with completion tracked.
📐 4. Draft Learning Objectives — Module 1

Based on the intake, the first module would address feedback delivery — a foundational, high-frequency leadership behavior tied directly to the Earn Trust LP.

  • 1
    Apply the SBI (Situation–Behavior–Impact) model to deliver specific, observable feedback in a real work conversation.
  • 2
    Distinguish between feedback that describes behavior versus feedback that judges character or intent.
  • 3
    Select the most effective response in a realistic feedback scenario involving a direct report.
  • 4
    Commit to a specific feedback conversation before the next 1:1 using a personal reflection prompt.
🏗️ 5. Proposed Solution — Modular Series

Given the need to build fast and scale, the recommended approach is a modular micro-learning series — each module 5–8 minutes, standalone, deployable to Amazon Learn independently, and expandable over time.

✓ Built

Module 1 — Delivering Feedback That Sticks

SBI model, Do/Don't comparison, branching scenario, personal reflection. Tied to Earn Trust LP.

📊 6. Measurement Plan — Kirkpatrick L1–L4
L1 Reaction End-of-module 3-question pulse in Amazon Learn. Tracks learner sentiment and perceived relevance.
L2 Learning Scenario-based knowledge check embedded in each module. Pass threshold tracked in Amazon Learn.
L3 Behavior 30-day follow-up survey: "Have you applied SBI feedback in the past 2 weeks? If not, what got in the way?"
L4 Results Correlate with existing people metrics (engagement scores, 360 feedback) once baseline data is available.
Day-one tracking: Completion rate, assessment scores, and L1 reactions can be live in Amazon Learn from the first module launch. L3 measurement can be set up with a single Qualtrics survey linked post-completion.
